Q: Why did we use “” here but not above?
  • Names of objects, functions, arguments, and special values (TRUE, FALSE, NA, NULL) should not be in quotes.
  • Other plain text should be in quotes.
  • Numbers should never be in quotes unless you want R to treat them as text.

Binary vs. non-binary variables

Binary Variables:

  • can take only two values: 1s and 0s

  • represent the presence/absence of a trait:

    • Example: voted= {1, 0, 0, 1, 1, 1, 0}
#Binary variables
voted <- c(1, 0, 0, 1, 1, 1, 0)

Non-binary variables:

  • can take more than two values
#Non-binary variables
distance <- c(1.452, 2.345, 0.298)
dice_roll<- c(2,4,6)
